# YoungMenTeamRed

This workspace contains a tiny single-file canvas game template for a quick 1-hour "vibe-code" session.

Files:
- [index.html](index.html) — single-file demo game (HTML + CSS + JS).

How to use:
- Open `index.html` in a browser (double-click or drag into browser).
- Click the "Start" button to begin. Move with arrow keys or WASD and collect the rotating squares.
- Edit `index.html` during the session: change colors, speeds, spawn rate, or replace mechanics.

Ideas for the group to try:
- Change the player shape, size, or add a dash/boost.
- Add enemies that chase the player.
- Give each collected item a random point value.
- Add sound effects using the Web Audio API or `<audio>`.
- Add levels, power-ups, or a high-score display using `localStorage`.

If you want, I can add starter challenges or a short comment-guided walkthrough inside `index.html`.
